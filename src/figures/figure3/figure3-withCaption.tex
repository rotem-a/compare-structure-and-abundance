  \captionof{figure}{\textbf{Differences in the dominant isoform between protocols.} \\
     a) Number of DTU genes and transcripts detected between direct RNA and PCR cDNA samples. Different colours indicate concordance between cell-lines. Light orange is the number of features that were DTU only in that cell line, and darker orange is the number of features detected in 5 or more of the cell-lines. b-d) Example of a DTU gene (SSNA1) where the dominant transcripts are different between protocols. b) Estimated proportions in direct RNA (left) and PCR cDNA (right), estimated using DRIMSeq from ‘bambu’ raw transcript counts for the same gene. c) Transcript counts (log2cpm) obtained by ‘bambu’ for this gene. d) Switching isoforms in SSNA1 compared: Dominant transcript in direct RNA is considered reference in black (ENST00000459860), dominant transcript in PCR cDNA is considered test in dark blue (ENST00000322310). In this case, the reference transcript is clearly longer than the test one. Structural differences between the two transcripts were identified, characterised, and then labeled on the test transcript with respect to the reference. Here an intron insertion was identified. While this difference was initially detected based on annotation and transcript quantification alone, coverage plots from representative samples provide qualitative validation of this based on aligned sequence data. Annotation plot of all transcripts detected in these samples by ‘bambu’ is provided for completeness. e) Fraction of DTU genes detected across cell lines, with the fraction of genes with dominant switching in darker grey. f) SQANTI3 labels obtained for each of the switching isoform pairs in each cell line. g) The number of structural differences detected between switching isoforms in each of the cell lines. h) Schematic representation of structural differences detected between reference (black) and test (purple) transcripts. i) Fraction of structural differences detected between reference and test transcripts in switching isoform genes (red), compared to the fraction of differences detected in sampled sets of background genes (grey).}
