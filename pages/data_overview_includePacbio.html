Author

Published

June 22, 2026

The data analysed here is publicly available from the SG-Nex database (Ying Chen et al. 2025). The SG-NEx data was accessed in June 2025 registry.opendata.aws/sg-nex-data. We started by downloading 107 .bam files (samples), which include read alignment to the human genome (hg38) for each of the replicates in each of the cell lines. ‘bambu’ v3.8.3 with NDR=0.1 was used to identify and quantify the genes and transcripts in all of the samples together. This resulted in a SummarisedExperiment R object, which includes transcript counts, and full length read counts as separate assays. Gene counts were obtained by using bambu::transcriptToGeneExpression, which effectively adds the transcript counts for each gene. For each sample, library size was obtained by adding up all transcript counts. Samples with very low or very high library size <400,000 or >40,000,000 were excluded, as well as those that had full length to all transcript ratio lower than 0.2. With the remaining 79 samples, MDS dimensional reduction plots were generated using edgeR::plotMDS for each of the assays independently, with top=2000, to account for the fact transcript counts are not independent of each other.

The data analysed here is publicly available from the SG-Nex database (Ying Chen et al. 2025). The SG-NEx data was accessed in June 2025 [registry.opendata.aws/sg-nex-data](https://registry.opendata.aws/sg-nex-data).
We started by downloading 107 .bam files (samples), which include read alignment to the human genome (hg38) for each of the replicates in each of the cell lines. 
'bambu' v3.8.3 with NDR=0.1 was used to identify and quantify the genes and transcripts in all of the samples together. This resulted in a SummarisedExperiment R object, which includes transcript counts, and full length read counts as separate assays. Gene counts were obtained by using bambu::transcriptToGeneExpression, which effectively adds the transcript counts for each gene. 
For each sample, library size was obtained by adding up all transcript counts. Samples with very low or very high library size  <400,000 or >40,000,000 were excluded, as well as those that had full length to all transcript ratio lower than 0.2. 
With the remaining 79 samples, MDS dimensional reduction plots were generated using edgeR::plotMDS for each of the assays independently, with top=2000, to account for the fact transcript counts are not independent of each other.
