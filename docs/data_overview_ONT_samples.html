SG-Nex data analysed here is publicly available from the SG-Nex database (Ying Chen et al. 2025). It was accessed in June 2025 at registry.opendata.aws/sg-nex-data. ONT samples only quantification started from 100 .bam files (ONT samples only), which include read alignment to the human genome (hg38) for each of the replicates in each of the cell lines. Then bambu (‘bambu’ v3.8.3 with NDR=0.1) was used to identify and quantify the genes and transcripts in all of the samples together.

This resulted in a SummarisedExperiment R object, from which this analysis page starts. It includes transcript counts and full length read counts as separate assays. Gene counts were obtained from it by using bambu::transcriptToGeneExpression() function, which effectively adds up all transcript counts for each gene.
