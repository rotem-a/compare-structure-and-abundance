About

Long read sequencing reveals changes in measured isoform structure and abundance due to RNA processing

These pages contain the code for the analysis and generation of figures discussed in our recent manuscript which can be referred to in this link.

Authors

Rotem Aharon1,2, Alicia Oshlack1,2,3

Abstract

Previously, RNA sequencing required Reverse Transcription (RT) and PCR amplification. PCR amplification is known to induce biases associated with expression levels, transcript lengths and GC content. Biases introduced by RT have been described more recently, and their quantification has been more elusive. Oxford Nanopore Technology (ONT) enables the direct sequencing of RNA without RT, and also the sequencing of cDNA without PCR amplification. This allows us to gain further insights into how different aspects of RNA preparation bias the data, with a new focus on isoforms structure and abundance.

Using seven cell line data sets from SG-NEx, we compare nanopore RNA sequencing of direct RNA (dRNA), direct cDNA (dcDNA) and PCR amplified cDNA (PCRcDNA). Across all the cell lines we identify hundreds of Differentially Expressed (DE) genes and isoforms, indicating a significant change in observed expression between the protocols. Furthermore, we identify Differential Transcript Usage (DTU) between protocols in hundreds of genes across cell lines, indicating significant changes in the measured isoform proportions.

Analysis of DE genes and isoforms in the context of expression levels, sequence length and GC content shows that, as expected, the largest number of DE isoforms is between the PCR samples and non-PCR samples. DE isoforms display known amplification bias with respect to expression level, sequence length and GC content. Interestingly we also observe decreased counts after PCR associated with higher levels of the m6A methylation mark on the native RNA.

We also analyse structural differences between isoforms that switch in dominance between direct RNA and PCR cDNA samples, and find that while some switching events relate to simple fragmentation/truncation of the RNA, most involve more complex structural differences including novel junctions, such as exon truncation, exon skipping and intron retention. When compared to differences found between background transcripts selected from non DTU genes, inserted introns (exitrons) are significantly more common in the isoforms after RT and amplification. Last, we find that visualisation of raw coverage data is a crucial step in the qualitative validation of experimental findings and should be included to validate analysis results.

### Reproducible Quarto Project with Anaconda and renv

- Raw data files or analysis output files are not included in this repository in order to reduce storage requirements.

### Additional software

This analysis further utilises the following: **bedtools**, **m6Anet**, **gffCompare**, **SQANTI3**.

- **Quarto**: For unified document creation with both languages

```bash
git clone [repo-url]

```

```bash
 conda env create -f environment.yml
 conda activate quarto-project
```

3. **Set up  R environment with renv**
```r
# Launch R in the project directory
renv::restore()
```
(or launch R with ' R -e "renv::restore()" ')
4. **Render the Quarto document**
Run this from the project's root directory where _quarto.yml is located.

```bash
quarto render
```
